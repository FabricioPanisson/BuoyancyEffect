# 🎮 Protótipo 2D — Movimento e Física com Água

Projeto desenvolvido na Unity com foco na implementação e demonstração de mecânicas básicas de jogos 2D.

---

## 🚀 Funcionalidades

* 🕹️ Movimento do player (A/D ou setas)
* 🦘 Sistema de pulo com detecção de chão
* 🌊 Simulação de água com Buoyancy Effector 2D
* 🧱 Colisão com chão utilizando layers

---

## 🧠 Objetivo

Demonstrar, de forma clara e funcional, o uso de física 2D na Unity, incluindo movimentação, pulo e interação com água.

---

## 🎮 Relação com jogos

* 🔹 Jogos indie: *plataformas 2D* como Terraria
* 🔹 Jogos AAA: sistemas de física e ambiente presentes em jogos como GTA V

---

## ⚙️ Funcionamento

* O movimento utiliza `Rigidbody2D` e input do teclado
* O pulo ocorre apenas quando o player está no chão (ground check)
* A água aplica forças físicas através do **Buoyancy Effector 2D**, simulando flutuação e resistência
